<h1> Library-Management-System</h1>
<p>  A simple Java console-based Library Management System built using the MVC (Model–View–Controller) design pattern.</br> This project is designed to run in Eclipse IDE and demonstrates core CRUD operations with a menu-driven interface.</p>
<h2>🔹 Features</h2>
<ul>
 <li> Add Book → Insert new books into the library collection.</li>
<li> Remove Book → Delete books by title.</li>
<li> Search Book → Find books and check availability.</li>
<li> Display All Books → List all books with status (Available/Borrowed).</li>
<li> Borrow Book → Mark a book as borrowed.</li>
<li> Return Book → Mark a borrowed book as returned.</li>
<li>Exit → Close the application gracefully.</li>
  </ul>
<h2>🔹 Project Structure</h2>
<ul>
<li>Model → Book.java (represents book entity with id, name, author, availability).</li> 
<li> Controller → Library.java (business logic for managing books).</li> 
<li> View → Main.java (menu-driven console interface for user interaction).</li> 
  </ul>
<h2>🔹 Tech Highlights</h2>
<ul>
<li> Written in Java (JDK 8+).</li>
<li>Runs in Eclipse IDE as a standard Java project.</li>
<li> Demonstrates Object-Oriented Programming (OOP) principles: encapsulation, abstraction, and modular design.</li>
<li> Menu-driven console interface for easy interaction.</li>
  </ul>
<h2>🔹 How to Run</h2>
<ul>
<li>Clone the repository.</li> 
<li> Open Eclipse → File → New → Java Project → Import the source files.</li> 
<li> Run Main.java.</li> 
<li> Use the console menu to interact with the library system.</li> 
</ul>
