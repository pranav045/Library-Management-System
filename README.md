# 📚 Library Management System (Java, MVC)

A simple **Java console-based Library Management System** built using the **MVC (Model–View–Controller)** design pattern.  
This project is designed to run in **Eclipse IDE** and demonstrates core **CRUD operations** with a menu-driven interface.

---

## 🚀 Features
- **Add Book** → Insert new books into the library collection  
- **Remove Book** → Delete books by title  
- **Search Book** → Find books and check availability  
- **Display All Books** → List all books with status (Available/Borrowed)  
- **Borrow Book** → Mark a book as borrowed  
- **Return Book** → Mark a borrowed book as returned  
- **Exit** → Close the application gracefully  

---

## 🏗️ Project Structure
- **Model** → `Book.java`  
  - Represents book entity with `id`, `name`, `author`, `isAvailable`  
- **Controller** → `Library.java`  
  - Business logic for managing books (add, remove, search, borrow, return, display)  
- **View** → `Main.java`  
  - Menu-driven console interface for user interaction  

---

## ⚙️ Tech Highlights
- Written in **Java** (JDK 8+)  
- Runs in **Eclipse IDE** as a standard Java project  
- Demonstrates **Object-Oriented Programming (OOP)** principles: encapsulation, abstraction, modular design  
- Menu-driven console interface for easy interaction
